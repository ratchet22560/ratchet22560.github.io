• These are some of the main web development terms with their defintions that could be worth knowing
  • HTML - Hypertext Markup Language, this is the language the computer uses to create websites
  • CSS - Cascading Style Sheet, this is the language that gives the website it's appearence in HTML, you can use it to change colours, location, text size and alignment using classes.
  • JavaScript - This is the most common programming language that HTML uses. It can be used to automate or animate something in a webpage, among other things. It has and can be used to make browser based games.
  • Operating System - This is what the computer uses to visualise processes the computer performs, each computer can have different operating systems, the main companies that provide them are, Microsoft which creates Windows, Apple which creates IOS and linux, which actually has a variety of different versions, which can be made by other companies. It is important to consider these when creating a website to tailor to them as there is a large market missed otherwise.
  • GitHub / Git - GitHub is a interface for Git, it allows for the user to have a type of backup that they can revert to, incase of any problems or damage someone could have caused,it also allows the user to work with others as well as maintain multiple projects./li>
  • Plugin - This could be considered an extention of an existing program, these can be used to add extra functionality that it would already have. Plugins are usually made by third-party organisations, plugins are useful, not only because they add extra functions, but because it can help you reduce the amount of memory being used on the website.
  • Wireframe - These are created as a method to determine what a website will look like before it is made, this can be helpful to visualise the layout of navigation and other objects.
